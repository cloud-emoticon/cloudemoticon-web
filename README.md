# cloudemoticon-web

Cloud Emoticon as a universal web application

## Prerequisites
* `Node.js`
* `yarn`

## Install dependencies
```bash
yarn install
```

## Run as webapp
```bash
yarn start
```

## Run as Electron app
```bash
yarn run start-headless
```
Then run
```bash
yarn run electron
```
