# cloudemoticon-web

Cloud Emoticon as a universal web application

## Prerequisites
* `Node.js`
* `yarn`

## Install dependencies
```bash
yarn install
```

## Run as webapp
```bash
yarn start
```

## Run as Electron app
```bash
yarn run start-electron
```

## Package Electron app locally
```bash
yarn run package-electron
```
